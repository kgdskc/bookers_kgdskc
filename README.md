# Bookers

## サイト概要
自分が読んだ本の感想を投稿したり、他者が読んだ本の感想を知ることができる。

### サイトテーマ
* 本の感想を投稿できる
* 新しい本を見つけるための手伝いができる

### ターゲットユーザ
* 学生
* 読書愛好家

### 主な利用シーン
* 気になっている本の評価が知りたい時
* 新しい作品に出会いたい時
